  # CrisisDumb

  A multi-tenant crisis management system built as a learning
  project.

  ## Tech Stack

  - **Monorepo**: PNPM + Turborepo
  - **Frontend**: React + Vite + TanStack Query
  - **Backend**: tRPC + Node.js
  - **Database**: PostgreSQL + Drizzle ORM
  - **Deployment**: SST (AWS Lambda)

  ## Project Structure

  crisisdumb/
  ├── apps/
  │   ├── web/              # React frontend
  │   └── api/              # tRPC backend
  ├── packages/
  │   ├── types/            # Shared Zod schemas
  │   └── db/               # Drizzle ORM setup
